# CakwnAI Auto Dashboard Generator 🚀📊

**Transform Any Dataset Into Intelligent Dashboards Instantly**

A futuristic AI platform that automatically converts datasets (CSV, Excel, JSON) into stunning, interactive dashboards complete with KPIs, Bar Charts, Pie Charts, and Line Charts, all powered by an intelligent Backend AI Engine.

## 🌟 Tech Stack
- **Backend**: FastAPI, Pandas, NumPy, Scikit-learn
- **Frontend**: React, TypeScript, TailwindCSS, Recharts
- **Design**: Premium Glassmorphism Cyber Theme

## 🛠️ How to Run

### 1. Backend API
```bash
cd backend
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
uvicorn main:app --reload --port 8000
```

### 2. Frontend UI
```bash
cd frontend
npm install
npm run dev
```

The platform will run on `http://localhost:3000`. Drag and drop any dataset to witness the AI auto-generate your entire dashboard!
